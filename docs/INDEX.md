# isx: The Python API to read Inscopix Data

`py_isx` is a lightweight pure-python API for reading 
Inscopix data files. 

## Support

Currently, `py_isx` supports reading CellSets. Support for other file types is planned.


|  File type | Support |
|  --------- | ------- |
| ISXD CellSet   | ✅ |
| ISXD Movie   | ✅ |
| GPIO data   | ❌ |
| ISXD Events   | ❌ |
| ISXD VesselSet   | ❌ |
